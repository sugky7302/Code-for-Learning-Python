符號:
  - #
    註解。

  - +-*/
    加減乘除。在運算結果最後加上「/1」，可使整數變成浮點數。字符串用「+」表連結之意。

指令:
  - %r
    輸出原始值。

  - %s
    輸出原始值內的數據。

  - %d
    輸出數字。

  - \n
    換行符號。

  - \t
    縮進符號。

  - def
    定義函數。函數內容將縮排在函數名底下，並以空格與非函數內容隔開。

  - return
    讓函數能夠返還數據。

函數:
  - print(字符串 % (參數), end = "")
    印出字符串。參數是字符串中有 %r %s %d 三種置換符號的時候使用。end後面接的是換行符號，可自定義兩個print之間是換行或是接續等等。

  - input(字符串)
    用戶輸入值，字符串可顯示在命令行中。

  - from A import B
    從A的包中引入B功能模塊。

  - open(檔名(字符串), 模式(字符串))
    打開檔案，目前都是txt檔。模式默認為 r:讀，其他為 w:寫 
  
  - read
    讀取檔案的全部內容。你可以把結果賦給一個變量。

  - close
    關閉檔案。跟編輯器的 文件-->保存 同意思。

  - readline
    讀取檔案中的一行。

  - readlines
    讀取檔案的全部內容，但以一行一行形成表格供讀取。

  - truncate
    清空檔案內容。

  - write(字符串)
    把字符串寫入檔案中。

  - seek(數字)
    移動檔案讀取指針到指定位置。

  - strip
    將字符串的首尾空白都去除。
